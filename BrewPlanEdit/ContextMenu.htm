Context Menu

Right clicking the Plan Directive List brings up a menu of operations.

Add

This leads to a series of directives which may be added. They are added with default values which can then be edited as desired.

I have grouped the directives into (possibly arbitrary) groups to make it quicker to access a given directive.

Add Alphabetical

This rearranges the directives into alphabetical order. It may be easier to find directives alphabetically, especially if you do not mentally group the directives the same way I do...

Comment/Uncomment Line

An existing directive may be converted to a comment:

#chill -10       ->        ;#chill -10

The comment may then be converted back to the directive with the Uncomment menu item.

;#chill -10       ->        #chill -10

Move Up/Down

These allow directives to be moved up or down in the plan.

Delete

This removes the directive from the plan. No, there is no Undo mechanism:)
